一个能够实现网页抽奖的小工具。

## 用法

1. 双击index打开配置页面，设置奖项、上传人员名单、设置抽奖配置。
2. 点击去抽奖，转跳到抽奖页面。进行抽奖。



## 功能

1. 对一些内容进行了抽象设计，不限定奖项人员。
2. 给出了一定程度的配置选项，便于使用。
3. 对配置信息进行本地保存，误操作友好。
